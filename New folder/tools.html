Upgrade Complete

  • Added support for duplicate bookmark ancestors.

This upgrade is intended to support the odd bookmark changes coming to Chrome.

Even though this version of Sprucemarks can sort two duplicately named locations that only differ by how they are being stored, Sprucemarks can not control how bookmarks are displayed. Expect browsers like Chrome to display things very strangely if you have a mix of local and synced bookmarks.

You can bypass all this confusion by only using local or synced bookmarks, and not mixing them both. The browser bookmark manager is a great location to audit and relocate bookmarks if you find yourself in this situation.

Google Chrome hides extensions by default but you can use the extensions menu in the upper right of Chrome to enable the pin option for Sprucemarks.

The above procedure will allow the Sprucemarks icon to be visible and enable one-click access.

Microsoft Edge hides extensions by default but you can use the extensions menu in the upper right of Edge to enable the eye option for Sprucemarks.

The above procedure will allow the Sprucemarks icon to be visible and enable one-click access.

Mozilla Firefox hides extensions by default but you can use the extensions menu in the upper right of Firefox to click on the gear for Sprucemarks which will then reveal a "Pin to Toolbar" option.

The above procedure will allow the Sprucemarks icon to be visible and enable one-click access.
